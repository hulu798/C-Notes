# 备选拼写：`iso646.h`头文件

> [!tip]
>
> 包含了`iso646.h`头文件可以用右边写法代替传统写法

| 传统写法 | ios646.h  |
| -------- | --------- |
| &&       | $and$     |
| &=       | $and\_eq$ |
| &        | $bitand$  |
| \|\|     | $or$      |
| \|       | $bitor$   |
| \|       | $or\_eq$  |
| ~        | $compel$  |
| !        | $not$     |
| !=       | $not\_eq$ |
| ^        | $xor$     |
| ^=       | $xor\_eq$ |
